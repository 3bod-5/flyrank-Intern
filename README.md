# Express.js Basic Server

A simple Express.js application that demonstrates how to create a web server with multiple routes.

## Features

- Express.js server
- Home route (`/`)
- Home page route (`/home`)
- Runs on port `3000`

## Project Structure

```
.
├── server.js
├── package.json
└── README.md
```

## Installation

1. Clone the repository.

```bash
git clone <your-repository-url>
```

2. Navigate to the project folder.

```bash
cd <project-folder>
```

3. Install dependencies.

```bash
npm install
```

## Running the Project

Start the server:

```bash
node server.js
```

The server will run on:

```
http://localhost:3000
```

## Available Routes

| Route | Description |
|-------|-------------|
| `/` | Displays "Welcome to my website" |
| `/home` | Displays "Welcome to my Home page" |
